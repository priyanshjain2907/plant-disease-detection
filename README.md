# 🌿 Plant Disease Detection System

## 📌 Overview

This project is a **Plant Disease Detection System** built using Deep Learning. It identifies plant diseases from leaf images and helps in early detection, which can improve crop health and yield.

The model is trained using image data and can classify different plant diseases with good accuracy.

---

## 🚀 Features

* 🌱 Detects plant diseases from images
* 🤖 Built using TensorFlow/Keras
* 📊 Image classification using CNN
* 📁 Easy-to-understand notebook implementation
* ⚡ Ready for further deployment (web/app integration)

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/plant-disease-detection.git
```

2. Open the notebook:

```
Plant_disease_detection.ipynb
```

3. Update dataset path in the notebook:

```
/path/to/your/dataset
```

4. Run all cells
